###변수의 타입확인

```
>>> type('안녕하세요')
<class 'str'>
>>> type(53)
<class 'int')
```
문자열은 str, 정수는 int

##문자열

###인덱스 연산
문자열에서 문자를 추출하기 위해 대괄호[]를 사용
가장 왼쪽은 0, 가장 오른쪽은 -1로 표현가능

```
>>>lux = '빛으로 강타해요!'
>>>lux[0]
'빛'
>>>lux[-1]
'!'
```
문자열은 불변, 새 값을 대입할 수 없다.

###슬라이스 연산
```
- [\:] 
    - 처음부터 끝까지
- [start\:]
    - start오프셋부터 마지막까지
- [:end]
    - 처음부터 end오프셋까지
- [start:end]
    - start오프셋부터 end오프셋까지
- [start\:end\:step]
    - start오프셋부터 end오프셋까지, step만큼씩 뛰어넘은 부분
```
